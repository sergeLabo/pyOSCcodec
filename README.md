Use decodeOSC(data) to convert a binary OSC message data to a Python list.

Use OSCMessage() and OSCBundle() to create OSC message.

OSCcodec.html in master directory

### Usage

##### Decode

Use decodeOSC(data) to convert a binary OSC message data to a Python list.

##### Create a message

Use OSCMessage() and OSCBundle() to create OSC message.
